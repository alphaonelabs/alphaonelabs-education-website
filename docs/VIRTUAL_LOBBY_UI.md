## Interactions

### Canvas (Left Panel)
- Click and drag: Move your avatar
- Real-time: See other users move
- Grid: Visual orientation

### Participants List (Top Right)
- Shows all active users
- Your avatar highlighted
- Display names and usernames

### Chat (Bottom Right)
- Send messages to all
- System notifications
- Join/leave alerts
- Message counter in header

## Responsive Design

### Desktop (> 1024px)
- 2-column layout
- Canvas 2/3 width
- Sidebar 1/3 width

### Tablet (768px - 1024px)
- 2-column layout
- Canvas 60% width
- Sidebar 40% width

### Mobile (< 768px)
- Single column
- Canvas full width
- Participants/chat below canvas
